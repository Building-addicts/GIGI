Stop any old server processes

Clears stale processes and lock files from previous sessions.

pkill -f "node server.js" 2>/dev/null; pkill -f "node panel.js" 2>/dev/null; rm -f "…/server/logs/bridge.lock"; echo "✓ Clean"
Expected output
✓ Clean
2
Navigate to the server folder

Open Terminal → paste this command.

cd "…/03_HARNESS/server"
3
Install dependencies

Only needed the first time — takes ~30 seconds.

One-time only
npm install

added N packages …
4
Start the server

Keep this Terminal window open — closing it stops the server.

node panel.js
