URL stabile permanente (`gigi.tuodominio.me`), funziona sempre, ri-pair mai.

  • 🎯 L'opzione definitiva: URL invariato anche dopo reboot PC
  • Richiede: account Cloudflare (gratis) + dominio aggiunto alla tua zone CF (~€5/anno)
  • Setup guidato in 3 passi, ~5 min una-tantum
