## What's intentionally NOT in v0.1.0

- Production-grade confirm gating UI (server can't emit `confirm_required`
  events yet — Claude CLI feature pending). ConfirmComputerUseSheet is
  wired client-side for the day Anthropic ships it.
- Multi-language UI (English only; harness LLM is multilingual but app
  surface is en-US for the global launch).
- Push notification proactive watchers (carryover from Phase 1, deferred).
- App Store distribution (Sideloadly only; App Store needs Family setup +
  Apple Intelligence entitlement reviews).
- Multi-user / family sharing (single-device pairing only).

These are all candidates for v0.2.0.
