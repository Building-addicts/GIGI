## 9. Stato architettura corrente (per orientarti)

Il main flow di `GigiAgentEngine.process()` è:

```
1. STT Apple Speech → text
2. memoria aggiorna user turn
3. #if DEBUG: Brain Path Override gate
   - auto → continua
   - appleFM → invoca GigiFoundationAgent.shared.process()
   - ollama → return stub "Path 3 not configured"
   - claude → invoca processForceClaude helper
4. Gate 1 — deterministicFastPath (NLU rule-based, 24 intent)
   - confidence ≥0.95 → GigiActionDispatcher.execute() → return
5. Gate 2 — harness Claude bridge
   - harness paired? → GigiClaudeBridge.shared.run() → return
   - non paired? → "Pair the harness from Settings" → return
```

NIENTE PIÙ Gate 3 Groq. Niente più Groq agentLoop. Niente più PlannerEngine.

Tutto il "ragionamento" oggi vive nel harness Claude (subprocess Claude Code via `GigiHarnessClient` → `claude-runner.js`). Quando GATE 2 lancia Apple FM router upfront, il flow diventa 5-path.
